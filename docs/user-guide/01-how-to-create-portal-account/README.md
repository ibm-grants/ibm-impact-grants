# How to Create a Portal Account

## Step 1 - Visit [ibm.versaic.com](https://ibm.versaic.com) in a web browser

![Step 1](images/impact-grant-login-page.png) 

## Step 2 - Click to <span style="color: blue">Create an account</span>

![Step 2](images/impact-grant-create-acct.png) 

## Step 3 - Enter a valid <span style="color: blue">**email address**</span> in both boxes. 

![Step 3](images/impact-grant-create-acct-email.png) 

## Step 4 - Check the "I'm not a robot" checkbox and answer the Captcha challenge.

![Step 3](images/impact-grant-create-account-captcha.jpg) 
![Step 3](images/impact-grant-create-account-captcha.png) 

## Click the <span style="color: blue">**Create new account**</span> button to proceed.
